## ğŸš€ Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/hippieZhou/hippiezhou.github.io.git
   cd hippiezhou.github.io
   ```

2. **Install dependencies**

   ```bash
   npm install
   cd src && npm install
   ```

3. **Start development server**

   ```bash
   cd src
   npm run server
   ```

4. **Build for production**
   ```bash
   cd src
   npm run build
   ```
